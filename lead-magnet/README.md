# The Shopify Developer Starter Kit (content)

Portable Markdown copies of the two written pieces of the kit, ready to drop into Notion or export to PDF:

- [`roadmap.md`](./roadmap.md) — How to Become a Shopify Developer (2026)
- [`cheat-sheet.md`](./cheat-sheet.md) — The Shopify Liquid Cheat Sheet (every line validated against the official Shopify docs)

The third piece of the kit is the Claude skills themselves, in the parent [`../skills/`](../skills/) directory.

On learnshopify.dev these render as hosted pages (`/starter-kit/roadmap`, `/starter-kit/cheat-sheet`) and are delivered by a one-time email when someone requests the kit. These files are the canonical source for the portable versions.
